# 接口 API

1. ###### 简介

   接口是前后端通信的桥梁

   简单理解：一个接口就是服务中的一个路由规则 根据请求相应结果

   接口通常称为API

   这里的接口指的是数据接口 与后端的接口语法不同

2. ###### 接口作用

   实现前后端通信

3. ###### 接口的开发与调用

   大多数接口都是由后端工程师开发 开发语言不限

   一般情况下接口都是由前端工程师调用 但有时后端工程师也会调用接口 比如短信接口 支付接口

4. ###### 接口的组成

   一般接口由以下几个部分组成

   - 请求方法
   - 接口地址
   - 请求参数
   - 响应结果

5. ###### RESTful API

   RESTful API是一种特殊风格的接口 主要特点如下

   - URL中的路径表示资源 路径中不能有动词 例如create delete update
   - 操作资源要与HTTP请求方法对应
   - 操作结果要与HTTP响应状态码对应
